## Request Parameters

<Tabs>
<TabItem value="required" label="Required" default>

- **symbol** `string`

  The ticker symbol.

</TabItem>
<TabItem value="date" label="Dates">

All `date` parameters are optional. By default, all stock split data is returned if no date parameters are provided.

- **from** `date`

  The start date for filtering stock splits (inclusive). Accepted timestamp inputs: ISO 8601, unix, spreadsheet.

- **to** `date`

  The end date for filtering stock splits (inclusive). Accepted timestamp inputs: ISO 8601, unix, spreadsheet.

</TabItem>
</Tabs>

## Response Attributes

<Tabs>
<TabItem value="Success" label="Success" default>
- **s** `string`

  Will always be `ok` when there is data for the stock splits requested.

- **date** `array[number]`

  Dates of the stock splits (Unix timestamp, America/New_York).

- **splitType** `array[string]`

  Type of the stock split (e.g., Forward, Reverse).

- **splitFrom** `array[number]`

  The number of shares before the split.

- **splitTo** `array[number]`

  The new number of shares after the split.

- **adjustmentFactor** `array[number]`

  The factor by which the number of shares was multiplied to achieve the new share count after the split. For example, if the adjustment factor is 2, it means the number of shares was doubled. If the adjustment factor is 0.5, it means the number of shares was halved.

- **updated** `number`

  The timestamp of the last update to our data for this ticker symbol (Unix timestamp, America/New_York).

</TabItem>
<TabItem value="NoData" label="No Data">

- **s** `string`

  Status will be `no_data` if no splits are found for the ticker symbol.

</TabItem>
<TabItem value="Error" label="Error">

- **s** `string`

  Status will be `error` if the request produces an error response.

- **errmsg** `string`
  An error message.

</TabItem>
</Tabs>
